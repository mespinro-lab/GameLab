# Life Tycoon 2 — Prototip

## Hipòtesi

L'eix d'inclinació impulsat per accions crea un bucle de joc auto-dirigit que se sent com una evolució natural del personatge, no com una elecció arbitrària. Els jugadors haurien de percebre com les seves accions modelen gradualment el personatge i desbloquegen noves possibilitats.

## Com executar

Obre `index.html` directament al navegador (no cal servidor — funciona des de `file://`).

No hi ha dependències externes.
